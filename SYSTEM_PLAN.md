# 銓宏國際 - 提案管理系統規劃書

**版本**: v2.0

**目標**: 建立專業的提案展示 + 後台管理系統

---

## 🎯 系統願景

### 核心價值
1. **給客戶看**：專業、美觀、易懂的產品提案展示
2. **給團隊用**：高效的提案製作、追蹤、管理工具
3. **數據驅動**：客戶行為分析，優化提案成功率

## 🗄️ 資料庫設計（Supabase）

### 核心資料表

```sql
-- 專案表
projects (
    id: uuid pk
    project_code: string unique
    customer_name: string
    contact_person: string
    business_model: enum ('一番賞', '企業禮贈', '活動贈品', '其他')
    quantity_range: string
    target_cost: string
    customer_budget: string
    payment_terms: string
    delivery_location: string
    status: enum ('draft', 'sent', 'feedback', 'approved', 'rejected', 'archived')
    created_by: uuid -> users
    created_at: timestamp
    updated_at: timestamp
)

-- 產品庫
product_library (
    id: uuid pk
    name: string
    category: string
    type: string
    base_cost: decimal
    suggested_quote: decimal
    specs: json
    features: text
    suppliers: json
    lead_time: string
    images: array
    tags: array
    is_active: boolean
)

-- 提案品項（專案與產品的多對多）
proposal_items (
    id: uuid pk
    project_id: uuid -> projects
    product_id: uuid -> product_library
    item_number: int
    custom_name: string
    custom_price: string
    custom_specs: text
    custom_features: text
    ichiban_level: enum ('A', 'B', 'C', 'D', 'E', '最後賞', '無')
    sort_order: int
)

-- 客戶回饋
client_feedback (
    id: uuid pk
    project_id: uuid -> projects
    interested_items: array -> proposal_items
    feedback_text: text
    contact_preference: string
    submitted_at: timestamp
    ip_address: string
    user_agent: string
)

-- 用戶
users (
    id: uuid pk
    email: string unique
    name: string
    role: enum ('admin', 'sales', 'designer', 'viewer')
    avatar_url: string
    last_login: timestamp
)

-- 操作日誌
activity_logs (
    id: uuid pk
    user_id: uuid -> users
    action: string
    entity_type: string
    entity_id: uuid
    details: json
    created_at: timestamp
)
```

## 🔧 功能模組詳細規劃

### 模組1: 提案範本系統

**目的**：快速建立專業提案

**範本類型**：
1. **一番賞範本**
   - 預設A/B/C/D/E賞結構
   - 自動計算數量分配
   - 包含一番賞專屬說明區塊

2. **企業禮贈範本**
   - 公司資訊區塊
   - 產品比較表格
   - 訂購流程說明

3. **活動贈品範本**
   - 活動主題風格
   - 預算導向分類
   - 交期倒數計時
